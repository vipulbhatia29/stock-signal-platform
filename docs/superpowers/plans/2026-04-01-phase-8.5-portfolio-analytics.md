### Task 2.3: New function `compute_quantstats_stock()`
- **File:** `backend/services/signals.py` (new function, add after `compute_risk_return()`)
- **Signature:** `def compute_quantstats_stock(closes: pd.Series, spy_closes: pd.Series, risk_free_rate: float = 0.045) -> dict`
- **Returns:** `{"sortino": float|None, "max_drawdown": float|None, "alpha": float|None, "beta": float|None}`
- **Implementation [REVIEW C2, C3, M2]:**
  ```python
  import quantstats as qs
  returns = closes.pct_change().dropna()
  spy_returns = spy_closes.pct_change().dropna()
  common = returns.index.intersection(spy_returns.index)
  if len(common) < 30:
      return {"sortino": None, "max_drawdown": None, "alpha": None, "beta": None}
  returns, spy_returns = returns[common], spy_returns[common]
  greeks = qs.stats.greeks(returns, spy_returns).to_dict()  # [C3] Series→dict
  return {
      "sortino": round(qs.stats.sortino(returns, rf=risk_free_rate), 4),  # [M2] pass rf
      "max_drawdown": round(abs(qs.stats.max_drawdown(returns)), 4),  # [C2] store positive
      "alpha": round(greeks.get("alpha", 0.0), 4),
      "beta": round(greeks.get("beta", 0.0), 4),
  }
  ```
- **Data dependency [REVIEW H1]:** Needs SPY price history. Add SPY to stocks table in migration (or ensure via seed). Fetch SPY closes from `stock_prices` once per pipeline run. If SPY has no data, skip QuantStats metrics (store null) — never fail the pipeline.

### Task 3.2: New function `compute_rebalancing()`
- **File:** `backend/services/portfolio.py` (new function)
- **Signature:** `async def compute_rebalancing(portfolio_id: UUID, strategy: str, db: AsyncSession) -> list[dict]`
- **Logic:**
  1. Fetch open positions for portfolio (tickers + current weights)
  2. Fetch 1y daily closes for all position tickers from `stock_prices`
  3. Build price matrix (DataFrame: date × ticker)
  4. If < 30 days or < 2 tickers: fall back to equal-weight (current logic)
  5. Compute expected returns: `expected_returns.mean_historical_return(prices)`
  6. Compute covariance: `risk_models.sample_cov(prices)`
  7. Strategy dispatch:
     - `min_volatility`: `EfficientFrontier(mu, S).min_volatility()`
     - `max_sharpe`: `EfficientFrontier(mu, S).max_sharpe()`
     - `risk_parity`: `HRPOpt(returns_df).optimize()` [REVIEW H2 — HRPOpt takes returns, not prices. Compute: `returns_df = prices.pct_change().dropna()`]
  8. Add constraints: `ef.add_constraint(lambda w: w <= max_position_pct / 100)` (or use `weight_bounds=(0, max_position_pct/100)` in constructor)
  9. `cleaned_weights = ef.clean_weights(cutoff=0.001)` [REVIEW L2 — 0.001 not 0.01, avoids zeroing small positions in 97-position portfolio]
  10. Calculate delta_shares and delta_dollars per ticker
  11. Return list of dicts for DB insertion
- **Error handling:** Catch `pypfopt.exceptions.OptimizationError`, solver failures → fall back to equal-weight
- **Cascade:** Import from `pypfopt` (EfficientFrontier, HRPOpt, expected_returns, risk_models, exceptions)
